The Smart Campus Study Room Booking system allows students to quickly check room availability, book study rooms through a website or app, receive confirmations and reminders, and access rooms using their student ID. It works on common devices like phones, laptops, and tablets and runs across different operating systems. The system uses Wi-Fi for student access and LAN for servers, with secure connections supported by firewalls, access control, and separate student/admin networks. TCP and IP handle data transfer and routing. In the EA view, the business layer manages bookings, the data layer stores user and room information, the application layer includes the booking system and notifications, and the technology layer includes servers and network infrastructure. Overall, it makes booking faster, easier, and more secure.
